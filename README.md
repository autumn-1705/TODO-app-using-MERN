# Todo App using MERN Stack

A simple Todo Application built using the MERN stack (MongoDB, Express, React, Node.js).  
This app allows users to add, view, complete, and delete tasks with real-time updates.

## Features
- Add new todos
- View all todos
- Mark a todo as completed
- Delete a todo
- Data stored in MongoDB
- REST API using Express & Node.js
- React frontend with state management using Hooks

## Tech Stack

### Frontend
- React
- JavaScript
- CSS
- Fetch API

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Setup & Installation

### Backend
1. Navigate to the backend folder:  
   cd backend
   
2. Install dependencies:
npm install

3.Start the server:
npm start


**###Frontend**

Navigate to the frontend folder:

cd frontend


Install dependencies:

npm install


Start the React app:

npm start


The frontend will run on http://localhost:4008.
